## Install env
Open folder in terminal
```bash
cd code/
```
2. Create virtual env
```bash
python3 -m venv venv/
```
3. Open virtual env
```bash
source venv/bin/activate
```
4. Install required dependencies
```bash
pip install -r requirements.txt
```
5. Close virtual env
```bash
deactivate
```
